# Product Demand Predictor

## Overview

The Product Demand Predictor is a full-stack web application that uses machine learning to estimate product demand based on key factors such as price, season, and marketing effort. The system helps users make better decisions related to inventory management and product planning.

## Objective

The main objective of this project is to reduce uncertainty in demand forecasting by providing a simple and user-friendly tool that generates predictions in real time.

## Features

* Predict demand (Low / Medium / High)
* Input parameters: price, season, marketing level
* Stores prediction history using MongoDB
* Interactive frontend for easy user input
* Backend API for processing predictions

## Tech Stack

* Frontend: React
* Backend: Flask (Python)
* Database: MongoDB
* Machine Learning: Scikit-learn

## Project Structure

* Backend: Handles API and ML model
* Frontend: User interface for input and results
* Model: Trained ML model for prediction

## How to Run

1. Start backend server:
   python app.py

2. Start frontend:
   npm install
   npm start
